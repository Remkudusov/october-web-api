ПРИ ОТЛАДКЕ САЙТА НА HEROKU!

Чтобы описанные ниже запросы сработали - при отправке их необходимо добавлять к текущму адресу в Heroku

Например, правильная ссылка для отправки GET-запроса на получение всех тестов будет выглядеть так: ваша www.your_link.com/get/tests/

sign_up/ - POST-запрос для регистрации пользователя. Тело POST-запроса:

После выполнения запроса необходимо авторизироваться в системе с помощью запроса log_in/.

log_in/ - POST-запрос для входа в систему. Тело POST-запроса:

После выполнения запроса необходимо взять из cookies значение параметра csrftoken и поместить в заголовок с названием X-CSRFToken. Проделывать данное действие необходимо после каждого нового входа пользователя в систему.

log_out/ - GET-запрос для выхода из системы.

tests/ - GET-запрос для получения всех тестов.

answer/ - POST-запрос для входа в систему. Тело POST-запроса:

Данный POST-запрос имеет возможность отправлять только авторизованный пользователь
