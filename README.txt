Инструкция по развёртыванию локально

Это докер контейнер. Для того, чтобы его развернуть, необходимо в терминале:
1) Перейти к корневой директории контейнера по команде cd C:/User/file/where/solution_factory
2) Далее написать команду "docker-compose up". Контейнер соберётся. 
После необходимо подождать примерно минуту и перейти по ссылке: http://localhost:8080.
В этой ссылке будет написана инструкция для API
